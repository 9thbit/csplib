Requries Python 3.3+, bibtex

To build
	make

To view locally:
	make
	make serve
	<open localhost:8000 in your browser>
	
To add a new problem use new_problem.sh or:
	add a new folder `probXXX` to problems/
	add specification.md
	add references.bib or references.html
	optionaly add models in probXXX/models/, data in data/ and results in results/

	see prob001, prob002, and prob005 for examples

Needs a reasonable shell such as  bash, zsh or tcsh
